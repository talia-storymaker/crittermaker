Work-in-progress. 3D character customization inspired by Nintendo's Animal Crossing franchise.

A [Next.js](https://nextjs.org/) web app using [Three.js](https://threejs.org/). Start with `npm run dev` or equivalent and open [http://localhost:3000](http://localhost:3000).
